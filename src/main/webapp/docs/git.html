Git & GitHub

Getting started with Git can seem daunting, but the following steps will guide you through the process:

  • 1. Download Git
  • 2. Install Git
  • 3. Configure Git
  • 4. Create a new repository
  • 5. Add files to the repository
  • 6. Commit changes
  • 7. Create a remote repository
  • 8. Connect the remote repository
  • 9. Push changes to the remote repository:

    • Download Git

    Download Git from the official website GitHub and follow the installation instructions

    url Operating system
    https://git-scm.com/download/mac MacOS
    https://git-scm.com/download/win Windows

    Installing Git

    Install Git using command line for different operating systems

    Command Operating system
    # apt-get install git Debian/Ubuntu
    # yum install git Red Hat Enterprise Linux, Oracle Linux, CentOS
    $ brew install git MacOS
    winget install --id Git.Git -e --source winget Powershell

    Configuring Git

    Open a terminal or command prompt and configure Git with your name and email address using the following commands.

    Command Description
    git config --global user.name [Your name] Set the name associated with your git commits.
    git config --global user.email [Your-email@example.com] Set the email associated with your git commits.

    OR

    Command Description
    git config --global --edit Edit the name associated with the git account.
    git config --global --list Display the current username and email associated with your git commits.

    Creating a Git Repository

    Create a new folder on your computer and navigate to it in the terminal. Use the following command to initialize a new Git repository

    Command Description
    $ git init Initialize a new git repository
    $ git clone [url] Clone/Copy a git repository from an online remote repository

    Adding files

    Adding your first file to your Git repository

    Command Description
    $ git add [filename] Add [filename] to staging area
    $ git add . Add all files to staging area

    Commit file

    Committing changes to the repository

    Command Description
    $ git commit Save changes to the local repository
    $ git commit -m Save changes to the local repository with a commit messages

    Example

    Connect remote respository

    Connect local repository to remote repository on git hosting services

    Command Description
    $ git remote add origin [remote-repository] Link local repository to remote repository

    Example

    Push to respository

    Push changes to the remote repository using the following command

    Command Description
    $ git push -u origin [remote-branch] Push changes to the remote repository
